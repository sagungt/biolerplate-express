# Experss Sequelize Boilerplate

## Usage

Install

`npm install`

Generate Migration

`npm run g:migration -- --name create-users-table`

Generate Model

`npm run g:model -- --name User --attributes name:string`

Generate Seed

`npm run g:seed -- --name user`

Migrate

`npm run migrate`

Undo Migrate

`npm run migrate:undo`

Seed

`npm run seed`

Undo Seed

`npm run seed:undo`
